# Hartman AI

AI consulting site. Vanilla HTML/CSS/JS. GitHub Pages.

## Stack

- `index.html` — single-page scroll
- `styles.css` — Palette A tokens (forest + terracotta), Cormorant Garamond + Nunito + DM Sans
- `mesh.js` — WebGL fragment-shader mesh background (hero)
- `main.js` — rotating tag, testimonial carousel, scroll-blur nav, mobile menu

## Development

Serve any static way:

```bash
npx http-server . -p 8765
```

## Deployment

Auto-published via GitHub Pages from `main` branch root. Custom domain `hartmanai.com` via `CNAME`.
